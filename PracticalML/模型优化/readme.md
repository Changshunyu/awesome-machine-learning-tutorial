# 数据使用

## 搜集与整理数据
  很多实际项⽬我们不会有现成的数据集，需要通过额外的⽅法获取和整理。因此项目的第一步就是准备好一个表情识别项⽬所需要的数据集，包括以下几部分：

- 使⽤爬⾍爬取图像和视频，从视频中提取图⽚
- 对获得的图⽚数据进⾏整理，包括重命名，格式统⼀，去重。
- 利⽤⼈脸检测算法删选出有⽤的样本，利⽤关键点检测算法裁剪出嘴唇区域。

(1)数据爬取
